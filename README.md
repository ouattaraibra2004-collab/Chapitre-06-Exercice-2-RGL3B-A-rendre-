# Devinette

Jeu de devinettes en Java 21. Le programme génère un nombre aléatoire entre 1 et 100, et le joueur a 10 essais pour le trouver.

## Prérequis

- Java JDK 21


## Structure

```
java/
├── src/
│   └── Devinette.java
├── .gitignore
└── README.md
```

## Exemple

```
Jeu de devinettes
Devinez un nombre entre 1 et 100. Vous avez 10 essais.

Essai 1/10 : 50
Trop grand !
Il vous reste 9 essai(s).

Essai 2/10 : 25
Trop petit !
Il vous reste 8 essai(s).

Essai 3/10 : 37
Felicitations ! Vous avez trouve le nombre 37 en 3 essai(s) !
```
